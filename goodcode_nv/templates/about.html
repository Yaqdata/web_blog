About this page

Goodcode website is set of code snippets, programming receipts, tips and tricks I've collected them over the years and thought that putting them on-line can be useful to somebody else. There is plan for larger open source sub-projects if time allows me. There will be no advertisements supporting this project. I'll try to add it to kick-starter maybe. There is strong emphasis on Linux, open source, Gnu and free as in freedom. Everything is open source here. Website source code can be downloaded from github (link in menubar). Log of progress can be viewed there.
